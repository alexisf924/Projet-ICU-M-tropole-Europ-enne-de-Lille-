# Projet-Corine-Land-Cover-d-partement-du-Nord-
This project shows all the computer processing carried out on various software programs (Geographic Information Systems with QGIS, Excel, Python by Google Collab) for a climatology project at the bachelor's degree level in geography and urban planning at the University of Aix-Marseille on the urban heat island phenomenon in the European Metropolis of Lille. 

Python file containing percentage statistics on major land-use categories from the 2012 CORINE Land Cover dataset for all municipalities in the Nord department

Python file containing statistics on the percentage distribution of major land-use categories from the 2012 CORINE Land Cover dataset for the city of Lille
